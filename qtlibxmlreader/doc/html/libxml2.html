libxml2

QtLibxmlReader requires the libxml2 library, version 2.6 or later. During development we used version 2.6.9.

If you are using Unix, the library is probably already installed on your system. Otherwise you can usually get binaries from your OS vendor or from http://www.xmlsoft.org/. Lastly, you can build it yourself from source from http://www.xmlsoft.org/.
